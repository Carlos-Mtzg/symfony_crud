<h1>Symfony CRUD Tutorial</h1>

Este repositorio contiene el código fuente de un CRUD (Create, Read, Update, Delete) en Symfony desarrollado como parte de un tutorial que publiqué en YouTube. En el tutorial, mostramos cómo construir un CRUD básico utilizando el framework Symfony.
<br><br><br>
**Requisitos**
- PHP 7.4 o superior <br>
- Composer <br>
- Symfony CLI <br>
- Un servidor web (como Apache o Nginx) <br>
- MySQL o cualquier otra base de datos soportada por Symfony <br>
